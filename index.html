MEDLIFE APP

About

Application allows one to track the day to day activities of the Student Organization.

Currently implemented features

  1. Login Page
    • Executive Members can login
  2. Events Tab:
    • Executives can see upcoming events hosted by the club
    • Designated executives can publish events in the app for others to see
  3. Task Tab
    • Executives can see tasks assigned to them
    • They can mark tasks as completed and upload screenshots as proof of task completion
  4. Executives Tab
    • Presidents can add or delete executives from the app
    • Presidents can assign or modify tasks assigned to other executive members
  5. Settings Tab
    • Executives can change their password and sign out

Running the App:

  1. Clone the repository
  2. Open project in xCode
  3. Hit Run

Simple Navigation Tool

About

Simple Navigation Tool is an application which allows one to visualize the shortest path between two points on the map of Edmonton, Alberta.

It uses dijkstra's algorithm to calculate the shortest path between two give coordinates. This information is then fed to a python plotter program using pipes. The Plotter then plots the shortest path on the map of Edmonton.

Since this project idea was partially inspired by a friend's assignment at the university of Alberta, I need to keep this repository private. But a video demonstratio of the project is given above. I am also willing to show the code at an interview or add the interested party as a collaborator in the Github repository.

GoGo- Better With You

___

About

GoGo is a website that connects individuals who want to explore events and experiences together while providing a platform for businesses to expand their reach.

GoGo was built because many individuals encounter the same problem - the desire to attend certain events but lacking companionship for the experience. These individuals represent a consumer segment that businesses often miss out on capturing. GoGo resolves this issue by allowing one individual to reach out to others in order to invite them to a specific event created by a business.

Since the project is undergoing development and is also part of our project idea for an ongoing course at the University of Toronto, the Github repository is private. But I am happy to show the code in an interview or add you as a collaborator if required. Nevertheless, a figma mockup of the initial project interface is given above.

Built with

  • MongoDB
  • Express
  • React
  • NodeJS

The MERN stack was chosen because it provides an easy way, fast way to develop a website and it is a widely popular tech stack with lots of tutorials and documentation that would make it easy for us to get started. Additionally, it has a strong developer community and a wealth of resources to solve commonly encountered issues during development.
